## ✨ Key Features

The system provides tailored experiences for three distinct user roles:

| Role | Responsibilities & Features |
| :--- | :--- |
| **🎓 Student** | View event schedules, submit project repos, track real-time scoring, and manage team profiles. |
| **👩‍🏫 Teacher / Judge** | Review project submissions, assign scores, provide feedback, and export rank lists. |
| **🛡️ Admin** | User management, event configuration (deadlines, categories), and high-level data analytics. |

---

## 🛠 Tech Stack

- **Frontend:** Pure HTML5, CSS3, and Vanilla JavaScript (ES6+). No heavy frameworks, ensuring `< 1s` initial load time.
- **Backend:** Node.js with Express-like routing logic.
- **Deployment:** Optimized for Vercel Serverless Functions.
- **UI/UX:** Responsive design with a clean, professional "Hackathon" aesthetic.
